2. ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กำหนดให้

hr&br, ul&ol, select&checkbox, radio&button, div&span, script&style

Tag Similarity Difference
hr&br hr&br are used to separate content by inserting a single line break hr tag displays a horizontal line break while br tag displays an empty line break
ul&ol ul&ol use the li tag to define list items. ul tag defines an unordered (bulleted) list while ol tag defines an ordered list. An ordered list can be numerical or alphabetical.
select&checkbox select&checkbox are selectable lists and always add the label tag for best accessibility practices! select tag is used to create a drop-down list while the checkbox is shown as a square box that is ticked (checked) when activated.
radio&button radio&button are clickable buttons radio created with the form tag and the input element let a user select ONE of a limited number of choices while inside a button tag you can put content, like text, images or link to another webpage.
div&span div&span are used as a container for other HTML elements to style them with CSS or to perform certain tasks with JavaScript. div tag is used to group multiline-elements while span tag is used to group inline-elements in a document.
script&style script&style are used to manipulate elements script tag is used to indirectly manipulate elements in a separate line by providing id while style tag is used to directly manipulate elements between the start style tag and the end style tag

3.ให้หาว่า input type มีอะไรบ้างแต่ละ type แตกต่างกันอย่างไร

Input type Description
input type="button" defines a button
input type="checkbox" defines a checkbox which let user select ZERO or MORE options of a limited number of choices.
input type="color" is used for input fields that should contain a color.
input type="date" is used for input fields that should contain a date.
input type="datetime-local" specifies a date and time input field, with no time zone.
input type="email" is used for input fields that should contain a e-mail address.
input type="file" defines a file-select field and a "Browse" button for file uploads.
input type="hidden" defines a hidden input field. A hidden field let web developers include date that cannot be seen or modified by users when a form is submitted.
input type="image" defines an image as a submit button. The path to the image is specified in the src attribute.
input type="month" allows the user to select a month and year.
input type="number" defines a field for entering a number.
Use the following attributes to specify restrictions:
  • max - specifies the maximum value allowed
  • min - specifies the minimum value allowed
  • step - specifies the legal number intervals
  • value - specifies the default value
Tip: Always add the label tag for best accessibility practices!
input type="password" defines a password field (characters are masked).
input type="radio" defines a radio button. Only ONE radio button in a group can be selected at the same time.
input type ="range" defines a control for entering a number whose exact value is not important. Default range is 0 to 100, but can be adjusted by the same attributes of input type="number".
input type="reset" defines a reset button which resets (deletes) all form values to its initial values.
input type="search" defines a text field for entering a search string.
input type="submit" defines a submit button which submits all form values to a form-handler.
Note: Remember to set a name for the search field, otherwise nothing will be submitted.
input type ="tel" defines a field for entering a telephone number.
input type ="text" defines a single-line text field. The default width of the text field is 20 characters.
input type ="time" defines a control for entering a time (no time zone)
input type="url" defines a field for entering a URL. The input value is automatically validated before the form can be submitted.
input type="week" defines a week and year control (no time zone)

Color

With CSS, color can be specified in different ways:

  • By color name
  • As RGB values
  • As hexadecimal values
  • As HSL values
  • As HWB values

Color Names

CSS supports 140 standard color names.

exp. Purple Lightblue Blue Green Yellow Orange Red

RGB Colors

An RGB color value is specified with: RGB (RedGreenBlue)

Each parameter defines the intensity of the color as an integer between 0 and 255. This makes 256*256*256 = 16,777,216 possible colors

Color RGB Color
RGB(255,0,0) Red
RGB(0,255,0) Green
RGB(0,255,0) Blue
RGB(0,0,0) Black
RGB(128,128,128) Gray
RGB(255,255,255) White

RGBA Colors

RGBA color values are an extension of RGB color values with an alpha chanel - which specifies the opacity for a color.

An RGBA color value is specified with: RGBA (Red Green BlueAlpha). The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

RGBA(255, 0, 0, 0.2) RGBA(0, 255, 0, 0.2) RGBA(0, 0, 255, 0.2)
RGBA(255, 0, 0, 0.4) RGBA(0, 255, 0, 0.4) RGBA(0, 0, 255, 0.4)
RGBA(255, 0, 0, 0.6) RGBA(0, 255, 0, 0.6) RGBA(0, 0, 255, 0.6)
RGBA(255, 0, 0, 0.8) RGBA(0, 255, 0, 0.8) RGBA(0, 0, 255, 0.8)

Hexadecimal Colors

A hexadecimal color is specified with #RRGGBB

The number of colors that can be represented by this system is 166 = 16,777,216

HSL Colors

HSL stands for hue, saturation, and lightness (ความเข้มสี, ความอิ่มตัวของสี, ความสว่าง).

HLS color values are specified with: hls (hue, saturation, lightness)

Hue is a degree on the color wheel from 0 to 360. 0 is Red, 120 is Green, 240 is Blue.

Saturation is a percentage value; 0% means a shade of gray and 100% is the full color.

Lightness is also a percentage; 0% is black, 100% is white.

Color HEX RGB Color HSL
#FF0000 RGB(255,0,0) Red HSL(0,100%,50%)
#00FF00 RGB(0,255,0) Green HSL(120,100%,50%)
#0000FF RGB(0,0,255) Blue HSL(240,100%,50%)
#000000 RGB(0,0,0) Black HSL(0,0%,0%)
#808080 RGB(128,128,128) Gray HSL(0,0%,50%)
#FFFFFF RGB(255,255,255) White HSL(0,0%,100%)
exp. background color is HLS(180, 50%, 50%) and letter color is HLS(0, 0%, 100%)

HSLA Colors

HSLA color values are an extension of HSL color values with an alpha channel - which specifies the opacity of a color.

An HSLA color value is specified with: HSLA (Hue, Saturation, Lightness, Alpha), where the alpha parameter defines the opacity. The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

HSLA(0, 100%, 30%, 0.3) HSLA(120, 100%, 30%, 0.3) HSLA(240, 100%, 30%, 0.3)
HSLA(0, 100%, 50%, 0.3) HSLA(120, 100%, 50%, 0.3) HSLA(240, 100%, 50%, 0.3)
HSLA(0, 100%, 70%, 0.3) HSLA(120, 100%, 70%, 0.3) HSLA(240, 100%, 70%, 0.3)
HSLA(0, 100%, 90%, 0.3) HSLA(120, 100%, 90%, 0.3) HSLA(240, 100%, 90%, 0.3)

Font Family

The font-family property speifies the font for an element.

The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.

In CSS, there are two types of font family names:

  • font family - a specific font family like "Times", "Arial", "Courier", etc.
  • generic family - a group of font families with a similar look like "Serif", "Sans-serif", "cursive", "fantasy" or "Monospace"

Start with the font you want, and always end with a generic family, to let the browser pick a similar font in the generic family, if no other fonts are available.

Note: Seperate each value with a comma.

Note: If a font name contains white-space, it must be quoted. Single quotes must be used when using the "style" attribute in HTML.

Font Variant

The font-variant property specifies weather or not a text should be displayed in a small-caps font.

In a small-caps font, all lowercase letters are converted to uppercase letters. However, the converted uppercase letters appears in a smaller font size than the original uppercase letters in the text.

exp. This is font-variant: normal;
exp. This is font-variant: small-caps;

Font Size

The font-size property sets the size of the text.

Being able to manage the text size is important in web design. However, you should not use font size adjustmant to make paragraph look like headings, or headings look like paragraphs.

Always use the proper HTML tags, like h1 to h6 for headings and p for paragraphs.

The font-size value can be an absolute, or relative size.

Absolute size:

  • Sets the text to a specified size
  • Does not allow a user to change the text size in all browsers (bad for accessibility reasons)
  • Absolute size is useful when the physical size of the output is known

Relative size:

  • Sets the size relative to surrounding elements
  • Allows a user to change the text size in browsers

Note: If you do not specify a font size, the default size for normal text, like paragraphs, is 16px (16px=1em).

Set Font Size With Pixels

Setting the text size with pixels gives you full control over the text size:

exp. font-size: 40px; font-size: 30px; font-size: 20px; and font-size: 10px;

Font-size: 40px; Font-size: 30px; Font-size: 20; Font-size: 10px;

Set Font Size With Em

To allow users to resize the text (in the browser menu), many developer use em insteas of pixels.

The em size is recommomended by the W3C.

1em is equal to the current font size. The default text size in browsers is 16px. So, the default size of 1em is 16px.

The size can be calculated from pixels to em using this formula: pixels/16=em

exp. font-size: 2.5em; font-size: 1.875em; front-size:1.25em; and font-size: 0.626;

Font-size: 2.5em; Font-size: 1.875em; Font-size: 1.25em; Font-size: 0.625;

Use a Combination of Percent and Em

The solution that works in all browsers, is to set a defult font-size in percent for the body element:

exp. font-size: 250%; font-size: 187.5%; front-size:125%; and font-size: 6.25%;

Font-size: 250%; Font-size: 187.5%; Font-size: 125%; Font-size: 6.25%;

Responsive Font Size

The text size can be set with a vm unit, which means the "viewport width".

That way the text size will follow the size of the browser window:

exp. font-size: 5vm; and font-size: 3vm;

Font-size: 5vw; Font-size: 3vw;

Font Shorthand

To shorten the code, it is also possible to specify all the individual font properties in one property.

The font property is a shorthand property for:
  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

exp. font: italic small-caps bold 2em arial, sans-serif;

Font: italic small-caps bold 2em arial, sans-serif;

exp. font: bold 1.2em 'Courier New', Courier, monospace;

Font: bold 1.2em 'Courier New', Courier, monospace;

exp. color: cadetblue; text-decoration:hotpink underline wavy; font: italic bold 1.5em Verdana, Geneva, Tahoma, sans-serif;

color: cadetblue; text-decoration:hotpink underline wavy; font: italic bold 1.5em Verdana, Geneva, Tahoma, sans-serif;

@font-face Rule

With the @font-face rule, web designers do not have to use one of the "web-safe" fonts anymore.

In the @font-face rule you must first define a name for the font (e.g. myFirstFont), and then point to the font file.
